Articles & Tutorials

 
  InstaMaterial concept (part 3) (frogermcs.github.io)

This post is part of a series of posts showing an Android implementation of Instagram with a Material Design concept.

 
Unpacking Fling Detection in Android Applications (room5.com)

Here's an article about using GestureDetector for fling detection.

 
RecyclerView Part 1: Fundamentals For ListView Experts (www.bignerdranch.com)

This article helps ListView experts make the transition to RecyclerView.

 
Be Reactive with Retrofit (prezi.com)

A presentation by Colin Lee on RxJava and how it's used by Retrofit, a type-safe REST client for Android and Java.

 
Building for Android TV — Episode 1 (medium.com)

This is the first of a series of technical articles that will deal with building apps for Android TV. The official documentation is still at an early stage and it doesn’t give hints on how to create a more custom experience. This articles will, hopefully, give you the missing bits.

 
Great animations with PageTransformer (medium.com)

Chris Basha writes about how to use PageTransformers to create a parallax effect in a ViewPager.

 
Reveal Activity Transitions (halfthought.wordpress.com)

a Software Engineer at Google writes on using the reveal transition with Lollipop Activity transitions.

 
Getting Started with Activity & Fragment Transitions - Part 1 (www.androiddesignpatterns.com)

This post gives a brief overview of Transitions and introduces the new Activity & Fragment transition APIs that were added in Android 5.0 Lollipop. This is the first of a series of posts Alex Lockwood will be writing on the topic

 

Design

 
  Daily Material Design Inspiration at MaterialUp (www.materialup.com)

Paper, Shadow and Ink - Inspiration for Material Design (papershadow.ink)

Taylor Ling, a Mobile Experience Designer and Google Developer Expert on UX/Design has launched a new blog for writing about the best examples of Material design.

 
Androidicons Goes Open Source (plus.google.com)

Opoloo decided to make their popular Android Developer Icons an open source project. You may now download the set for free and do with it whatever you like, including the icon font.

Libraries & Code

 
ScrollableItemList (github.com)

An Android RecyclerView List with items that can scroll to reveal more content on the side. This view has been inspired by the one found in the Reddit News App

 
Grab-n-Run (github.com)

Grab’n Run (aka GNR) is a simple and effective Java Library that you can easily add to your Android projects to perform secure dynamic class loading operations over standard DexClassLoader

 
DiscreteSeekBar (github.com)

Here's the start of a project on GitHub to implement Material design's Discrete Slider. The author is looking for help in adding polish and publishing to jcenter.

 
BabushkaText (github.com)

BabushkaText is a custom TextView which lets you customize the styling of parts of your text via Spannables, but without the hassle of having to deal directly with Spannable themselves.

 
Spotify Android SDK (developer.spotify.com)

The Spotify Android SDK makes it easy to add audio streaming, user authentication and other Spotify features to Android apps.
